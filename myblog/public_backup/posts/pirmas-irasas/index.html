Štai paprasta Java klasė, atspindinti laivo objektą:

public class Laivas {
    private String pavadinimas;
    private int talpa;
    
    public Laivas(String pavadinimas, int talpa) {
        this.pavadinimas = pavadinimas;
        this.talpa = talpa;
    }
    
    public String getPavadinimas() {
        return pavadinimas;
    }
    
    public int getTalpa() {
        return talpa;
    }
}
